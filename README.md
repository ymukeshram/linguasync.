# LinguaSync – AI-Powered Translation Platform

LinguaSync is a modern AI-powered web application that enables users to **translate text, voice, and images across multiple languages** in real-time. It also includes a **chat system** and **OCR capabilities** to break language barriers seamlessly.

---

##  Features

*  **Multi-language Translation**
*  **Voice Input Support**
*  **OCR (Image to Text Translation)**
*  **Translation History**
*  **Fast AI-powered responses (Gemini API)**
*  **Modern Dark UI with Glassmorphism**

---

##  Tech Stack

### Frontend

* React + Vite
* TypeScript
* CSS (Modern UI)

### Backend

* Node.js
* Express.js
* Socket.io (Real-time communication)

### AI Integration

* Google Gemini API (`@google/genai`)

## 🧠 How It Works

* User enters text / voice / image
* Backend sends request to Gemini API
* AI processes and returns translated result
* Real-time chat handled via Socket.io
